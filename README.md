# PDF Page Size Resizer

A pure frontend PDF page resizing tool built with Vue 3 + Vite + TypeScript + Tailwind CSS v4.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- PDF processing is done locally in-browser with `pdfjs-dist` and `pdf-lib`.
- No backend API is required.
